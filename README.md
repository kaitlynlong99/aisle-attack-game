# Aisle Attack
Aisle Attack is a 2D arcade-style game built in Unity where the player controls a shopping cart, collects good items, and avoids bad items.
The game includes score tracking, level progression, and multiple UI screens for starting, restarting, and ending the game.

# Technologies Used
- Unity
- C#

## Features
- Player-controlled shopping cart movement
- Collision detection for collecting and avoiding items
- Score tracking system
- Level progression with increasing difficulty
- Start screen, game over screen, and restart functionality
- Audio and UI elements for gameplay feedback

## My role 
I designed and developed the game mechanics, implemented the scripting in C#, and built the gameplay flow in Unity

## Project Purpose
This project was created to practice game development concepts such as object interaction, event handling, UI design, and scripting in Unity.
